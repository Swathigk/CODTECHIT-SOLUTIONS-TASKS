##In this project, I created a Smart Light Control project on the Arduino platform in the Tinkercad simulation environment. Tinkercad is a web-based application that allows you to design, test, and run circuits online without the need to pick up any spare hardware. I coded the sketch in the IDE of Tinkercad in text mode and simulated the entire process. The circuit was arranged using an Arduino UNO, a handful of LEDs, 220- ohm resistors, a breadboard, and some jumper wires. The LEDs are connected to the resistors to ensure that the current does not burn them but instead, they remain safe. The LEDs contain an anode (positive) and a cathode (negative). I attached the anode to a digital pin using a resistor and carried the cathode to the GND pin to complete the circuit. The code is developed in the Arduino C/C++. I have stated three variables: led1, led2 and fan that will be mapped to pin 13, pin 12 and pin 8 respectively. I configured every pin as OUTPUT in the setup () function using pinMode. Within loop() I coordinate the devices sequentially. The LED1 will blink one second and then switch off. Next, LED2 does the same. At last, the fan heats two seconds and switches off. This sequence is an endless loop, since it is located within loop. I tried the Serial Monitor in Tinkercad previously. It simply allows the computer to communicate with the Arduino. A 1 would switch the LED on and a 0 would switch it off. This demonstrates a basic type of digital command control. The entire practice provided me with a good understanding of the basics of Arduino, including basic wiring and serial I/O. It is a simple demonstration of the functioning of smart light or device control. Such systems can be easily expanded to home automation, smart classroom systems, or entry-level IoT projects where you desire to be able to control lights or appliances with a remote source.
Overall, it was a valuable practical experience of learning how to use microcontrollers to control electronic devices, and it actually cemented my practical knowledge of Arduino.
